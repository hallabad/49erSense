This is the client for the IOT final project that simulates a node connecting to the server.

To run this:
	- Make sure the server is running
	- Start the server with the command "java -jar IOTSimClient.jar"

To use:
	- enter a command like "set-door closed [ID]" wher [ID] is the name of the door.
	- The server will show that the command was receved and how the command was exicuted.
	- To exit the client and close the connection, enter "e" into the consol.
